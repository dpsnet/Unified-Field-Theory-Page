## 部署方法

1. **克隆仓库**
   ```bash
   git clone https://github.com/dpsnet/Fractal-Clifford-Algebra-UFT-Page.git
   ```

2. **本地预览**
   - 直接在浏览器中打开 `index.html` 文件

3. **GitHub Pages 部署**
   - 将仓库推送到GitHub
   - 在仓库设置中启用GitHub Pages
   - 选择 `main` 分支作为源
